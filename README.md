# DESAFIO QA BEEDOO 2026

## Análise Inicial da Aplicação

A aplicação analisada tem como objetivo permitir o cadastro e a visualização de cursos por meio de uma interface web simples.

O sistema possibilita que usuários registrem cursos informando dados como nome, descrição, instrutor, período de realização, número de vagas e tipo do curso, além de visualizar os cursos cadastrados em uma listagem.

---

## Principais Fluxos Identificados

Durante a exploração inicial, foram identificados os seguintes fluxos principais:

- Cadastro de cursos
- Listagem de cursos cadastrados
- Navegação entre páginas do sistema

---

## Pontos Críticos para Teste

Os pontos considerados mais críticos para validação são:

- Formulário de cadastro de curso
- Validação de campos obrigatórios
- Validação de datas (início e fim)
- Inserção de dados inválidos
- Atualização correta da listagem após cadastro
- Integridade das informações exibidas

---

## Estratégia Inicial de Testes

A estratégia adotada foi baseada na análise funcional da aplicação, priorizando o fluxo principal de cadastro de cursos, seguido por cenários negativos e validações de entrada de dados, considerando possíveis erros de uso por parte do usuário.
